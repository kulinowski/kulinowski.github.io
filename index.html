Bloxorz

I created my own version of Bloxorz. It was a popular browser game in middle school, and if you're curious to play the original, then I added a link for you below. The aim of the game is to get the block to fall into the square hole at the end of each level. Use the arrow keys or WASD to move the block around the level. If the block falls off an edge, then the level will be restarted.
